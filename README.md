# 📚 API da Biblioteca

API REST desenvolvida em ASP.NET Core para gerenciamento de uma biblioteca, com autores, livros e empréstimos.

---

## 🚀 Tecnologias
- ASP.NET Core
- C#
- Swagger (OpenAPI)

---

## 📌 Funcionalidades
- CRUD de Autores
- CRUD de Livros
- Empréstimos e devoluções
- Paginação e filtros

---

## 📄 Documentação
Swagger disponível para testes da API.

---

## ▶️ Como testar a API

Rodar o projeto:
```bash
dotnet run
````

Acessar Swagger:

```
http://localhost:5102/swagger
```

Ou acessar diretamente:

```
http://localhost:5102/api/autores
```
